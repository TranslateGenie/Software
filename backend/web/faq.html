Frequently Asked Questions

Everything you need to know about Translate Genie.

How do I install Translate Genie?

Translate Genie is distributed through GitHub Releases. Head to the Download page, click Download Latest Release, and run the .exe installer. Once installed, launch the app from your Start Menu or Desktop and enter your license key to activate.

What document types can I have translated?

Translate Genie supports a wide range of file types across several categories:

  • Office documents — DOCX, PPTX, XLSX
  • Markup & web — HTML, XML, SVG, Markdown
  • Stylesheets — CSS, SCSS, SASS, LESS
  • Code files — JS, TS, Python, Ruby, PHP, Java, C#, Go, Rust, Swift, and more
  • Data & config — JSON, YAML, TOML, CSV, INI, and more
  • Subtitles — SRT, VTT
  • Plain text — TXT, LOG, RTF, and others

PDF files are not currently supported for translation.

Can Translate Genie translate embedded text in images?

Not currently. Translating text embedded in images requires optical character recognition (OCR) followed by image reconstruction to re-embed the translated text — a process that is complex and often produces inconsistent results. This is not supported at this time. If you need to translate image-based documents, consider converting them to a text-based format first.

Can Translate Genie translate text in videos?

Yes — through subtitle files. Translate Genie supports SRT and VTT subtitle formats. Upload your subtitle file, select a target language, and the app will translate all the dialogue lines while preserving your original timestamps exactly. You can then load the translated subtitle file into your video player or editor.

Is there a free trial?

There is no free trial at this time. However, you can get started with our smallest pack — One Wish — at just $10 for 10 requests or 200,000 characters, which is plenty to evaluate the quality of translation for your use case. Visit the Pricing page to see all available options.
