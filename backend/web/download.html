Installation Guide

  1. Click Download Latest Release above and download the .exe installer from the GitHub release page.
  2. Optionally verify the SHA-256 checksum listed on the release page against the downloaded file.
  3. Run the installer. Windows may show a SmartScreen prompt — click More info → Run anyway. This is expected for a new publisher certificate.
  4. Launch Translate Genie from your Start Menu or Desktop.
  5. Enter your license key when prompted to activate the app.

System Requirements

  • Windows 10 or Windows 11 (64-bit)
  • ~200 MB disk space
  • Internet connection required for translation (Azure AI Translator)
  • A valid Translate Genie license key — purchase one here
